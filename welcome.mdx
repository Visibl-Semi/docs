# Welcome to Visibl

Welcome to Visibl documentation! We're excited to help you create beautiful, functional documentation for your projects.

## What is Visibl?

Visibl is a modern documentation platform that makes it easy to create, maintain, and share technical documentation. Built with developers in mind, it offers:

<CardGroup cols={2}>
  <Card title="Easy to Use" icon="rocket">
    Simple markdown-based content creation with powerful components
  </Card>
  <Card title="Beautiful Design" icon="palette">
    Clean, modern design that looks great out of the box
  </Card>
  <Card title="Developer Friendly" icon="code">
    Git-based workflow with version control and collaboration
  </Card>
  <Card title="Fast Performance" icon="bolt">
    Lightning-fast page loads and search functionality
  </Card>
</CardGroup>

## Key Features

- **Markdown Support** - Write documentation in familiar Markdown syntax
- **Component Library** - Use pre-built components for rich content
- **API Documentation** - Auto-generate API docs from OpenAPI specs
- **Search** - Built-in search to help users find what they need
- **Customizable** - Tailor the look and feel to match your brand
- **Analytics** - Understand how users interact with your docs
