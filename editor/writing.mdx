# Writing

Learn how to create effective documentation content using Visibl's writing features.

## Markdown Basics

Visibl uses standard markdown syntax with additional features for technical documentation.

### Headers

```markdown
# H1 Header
## H2 Header  
### H3 Header
```

### Text Formatting

```markdown
**Bold text**
_Italic text_
`Inline code`
```

### Lists

```markdown
- Unordered list item
- Another item

1. Ordered list item
2. Another item
```

## Advanced Features

### Code Blocks

Use code blocks with syntax highlighting:

```javascript
function example() {
  console.log("Hello, Visibl!");
}
```

### Links and References

```markdown
[Link text](https://example.com)
[Internal link](/other-page)
```

## Best Practices

<Tip>
Keep your content clear and concise. Use headings to structure your content hierarchically.
</Tip>

- Use descriptive headings
- Keep paragraphs short
- Include examples where helpful
- Use lists for better readability
