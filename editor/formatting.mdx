# Formatting

Learn about text formatting options available in Visibl.

## Basic Formatting

### Text Styles

- **Bold text** - Use `**text**` or `__text__`
- *Italic text* - Use `*text*` or `_text_`
- `Inline code` - Use backticks
- ~~Strikethrough~~ - Use `~~text~~`

### Code Formatting

```python
def hello_world():
    print("Hello, Visibl!")
```
