## Administrator Setup (One-Time)

### Install latest version to a custom path (recommended)

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash -s -- --install-dir /project/tools/visibl
```

This will:
- Download the latest Visibl AppImage automatically
- Install to your specified path
- Create system-wide PATH via `/etc/profile.d/`
- Create symlinks in `/usr/local/bin/` for immediate access

### Install latest version (default path)

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash
```

Installs to `/opt/visibl` by default.

### Install a specific version (optional)

If you need to pin a specific version:

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash -s -- --version <version> --install-dir /project/tools/visibl
```

For example:

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash -s -- --version 2.0.4 --install-dir /project/tools/visibl
```

### Alternative: AppImage-Only Install (Per-Machine or Per-User)

You can also run Visibl directly from the AppImage, without using the installer:

1. Go to the [Visibl Releases page](https://github.com/Visibl-Semi/Releases/releases)
2. Download the desired `Visibl-<version>-linux-x64.AppImage`
3. Make it executable and run it:

```bash
chmod +x Visibl-2.0.4-linux-x64.AppImage
./Visibl-2.0.4-linux-x64.AppImage
```

You can place the AppImage wherever your team prefers (e.g., a shared tools directory or a user's home directory) and create a desktop shortcut or alias if desired.

## Launch Visibl

Once installed via the admin installer, any user can launch Visibl:

```bash
vs
```

Or:

```bash
visibl
```
