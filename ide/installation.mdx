## Administrator Setup (One-Time)

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash -s -- --install-dir /project/tools/visibl
```

This downloads the latest Visibl, installs it to your specified path, and configures system-wide access for all users.

### Options

| Option | Description | Example |
|--------|-------------|---------|
| `--install-dir <path>` | Custom install location | `--install-dir /project/tools/visibl` |
| `--version <version>` | Install an older version | `--version 1.9.4` |
| *(no arguments)* | Installs latest to `/opt/visibl` | `sudo bash` |

### Manual AppImage Install

Alternatively, download and run the AppImage directly:

```bash
curl -L -O https://github.com/Visibl-Semi/Releases/releases/download/linux-2.0.5/Visibl-2.0.5-linux-x64.AppImage
chmod +x Visibl-2.0.5-linux-x64.AppImage
./Visibl-2.0.5-linux-x64.AppImage
```

## Launch Visibl

Once installed via the admin installer, any user can launch Visibl:

```bash
vs
```

Or:

```bash
visibl
```
