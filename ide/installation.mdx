## Administrator Setup (One-Time)

Administrator downloads Visibl IDE:

```bash
cd ~/Downloads
curl -L -O https://github.com/Visibl-Semi/Releases/releases/download/linux-2.0.4/Visibl-2.0.4-linux-x64.AppImage
```

Or download from: `https://www.visiblsemi.com/enterprise`

Administrator runs global installation:

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash
```

This will:
- Locate the downloaded AppImage in `~/Downloads`
- Install Visibl to `/project/tools/visibl` (or custom path)
- Create global launch commands
- Set up user configuration script

## User Setup

Each user runs the setup script:

```bash
/project/tools/visibl/bin/visibl-setup-user
```

This will:
- Add Visibl to your PATH
- Configure application menu entry

Launch from a new terminal:

```bash
vs
```

Or launch with:

```bash
visibl-gui
```
