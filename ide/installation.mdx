## Administrator Setup (One-Time)

Install the latest version:

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash
```

Install a specific version:

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash -s -- --version 1.9.4
```

Install to a custom path:

```bash
curl -s https://www.visiblsemi.com/api/visibl-installer/admin-install | sudo bash -s -- --install-dir /project/tools/visibl
```

This will:
- Download the Visibl AppImage automatically
- Install to `/opt/visibl` (or custom path)
- Create system-wide PATH via `/etc/profile.d/`
- Create symlinks in `/usr/local/bin/` for immediate access

## Launch Visibl

Any user can launch immediately:

```bash
vs
```

Or:

```bash
visibl
```

Optional desktop integration (run as user, not sudo):

```bash
visibl-setup-desktop
```
