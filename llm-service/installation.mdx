## Install

Install the LLM service on your server:

```bash
curl -fsSL https://visiblsemi.com/api/install-llm | sudo bash
```

The installer automatically:
- Validates system requirements
- Installs NVIDIA Container Toolkit
- Downloads and installs the default model
- Starts service on port 58180

## Verification

```bash
visibl-smi
```

You should see your GPU(s) with the running Visibl LLM service.

## Configuration

### Custom Port

```bash
export SERVICE_PORT=8080
curl -fsSL https://visiblsemi.com/api/install-llm | sudo bash
```

## Troubleshooting

If issues occur, send us:

```bash
# Service logs
visibl-llm logs -f

# GPU status
visibl-smi
```

Email output and description to [hi@visiblsemi.com](mailto:hi@visiblsemi.com).
