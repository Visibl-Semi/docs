## Core Features

**Domain Expertise**: SystemVerilog/Verilog/VHDL, UVM, industry protocols (AXI, PCIe, DDR), EDA tool flows.

**Local Processing**: All code and designs stay on your infrastructure. Zero external API calls.

**Project Context**: Understands your codebase, build flows, and coding standards via `AGENTS.md`.

## Capabilities

**RTL Design**: Module generation, state machines, datapaths, clock domain crossing, protocol interfaces

**Verification**: UVM testbenches, SVA properties, functional coverage, formal properties

**Documentation**: Automated RTL documentation with timing diagrams and interface tables

**Analysis**: Code review, timing optimization, lint violations, protocol compliance

**Supported Tools**: Synopsys (VCS, DC), Cadence (Xcelium, Genus), Mentor (Questa), open source (Verilator, Yosys)

## Usage

**Initialize your project**
```bash
visibl init
```

Creates `AGENTS.md` with context about your build flows, EDA tools, and coding standards.

**Interactive TUI**
```bash
visibl
```

**Direct Commands**
```bash
visibl run "Create an AXI4-Lite slave interface"
```

**Documentation**
```bash
visibl docs uart_core
```

## Architecture

The CLI auto-discovers and connects to your local LLM service:

```bash
visibl --version
> Connected to: visibl-llm@127.0.0.1:58180
```

All AI processing happens on your infrastructure. Manual endpoint configuration available if needed.
