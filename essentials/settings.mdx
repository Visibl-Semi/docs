Learn how to customize your documentation site through various configuration options.

## mint.json Configuration

The `mint.json` file is the main configuration file for your documentation:

```json
{
  "$schema": "https://mintlify.com/schema.json",
  "name": "Your Documentation",
  "colors": {
    "primary": "#000000",
    "light": "#000000",
    "dark": "#ffffff"
  }
}
```

## Color Customization

### Primary Colors

Set your brand colors:

```json
"colors": {
  "primary": "#000000",
  "light": "#000000", 
  "dark": "#ffffff",
  "anchors": {
    "from": "#000000",
    "to": "#333333"
  }
}
```

### Background Colors

Mintlify automatically handles light and dark mode backgrounds, but you can customize accent colors.

## Logo and Branding

### Logo Configuration

```json
"logo": {
  "dark": "/logo/dark.png",
  "light": "/logo/light.png"
}
```

### Favicon

```json
"favicon": "/favicon.svg"
```

## Navigation Structure

Configure your sidebar navigation:

```json
"navigation": [
  {
    "group": "Get Started",
    "pages": [
      "introduction",
      "quickstart"
    ]
  }
]
```

## Advanced Settings

### Custom Domain

You can host your docs on a custom domain through the Mintlify dashboard.

### Analytics

Integrate with Google Analytics, Mixpanel, and other analytics tools.
